# EcoMine

A local-first eco-themed tap-to-earn crypto simulation app built with Flutter 3.24+, Clean Architecture, Riverpod, Freezed, and Hive.

## Architecture
- **Clean Architecture:** Divided into `core`, `data`, `features`, and `routing`.
- **State Management:** Riverpod 2.5+
- **Data Models:** `freezed` + `json_serializable` for immutability.
- **Persistence:** Local persistence using `hive`.
- **Routing:** `go_router` for structured navigation.
